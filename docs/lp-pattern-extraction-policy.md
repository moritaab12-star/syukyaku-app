# LP 構造抽出 — データ保持方針（著作権・個人情報）

Phase 3 `lib/extract/` は **ランディングページの「構造メタ」**のみを生成する。

## 保持するもの（許可）

- セクション役割の抽象ラベル（`RecommendedSectionRole` の列）
- CTA 種別の列挙（`tel` / `form` / `line` 等）
- 信頼ブロックの**種類ラベル**（`years` / `cases` 等の短いキー）
- 見出しレベルの最大値、ベネフィット候補ブロックの**個数**
- ソース URL・抽出時刻

## 保持しないもの（禁止）

- **本文の長文**（サービス説明の段落、お客様の声の全文、記事本文など）を DB・ログ・API レスポンスに永続化しない。
- 抽出処理の過程で見出しテキストを読む場合は **メモリ上のみ** とし、**1 見出しあたり最大 120 文字**に切り詰め、出力オブジェクト（`ExtractedPattern` / `PatternConsensus`）には**含めない**。
- スクレイプした HTML 全文をファイルに書き出したり、検証以外で再配布しない。

## 実装ノート

- 取得は `fetchHtmlLight` と同様の**先頭バイト上限**で行い、ページ全体のコピーとならないよう制限する。
- デバッグ時はローカルのみ・マスク推奨。
